Configuration of DUPAY
The DUPAY API is centered around REST. The API is constructed to have predictable, resource-oriented URLs as well as to use HTTP response codes to point out API errors. JSON will be returned in all responses from the API, including errors.
To get the app key, one needs to be registered as Merchant first.
After logging in, the app key will be found in Merchant Profile page.
Step One: Create Invoice
This generate an invoice for the transaction.
URL:http://103.221.253.163:8080/api/external/invoice
Method: POST
Request body should be valid JSON format. The parameter list in the body is stated below.
appKey Compulsory
This token uniquely identifies the ecommerce site of the merchant
Amount Compulsory
The amount you want to charge for this invoice.
redirectUrl Compulsory
Redirect url after payment.
